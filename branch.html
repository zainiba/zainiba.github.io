Generated Branches

Maya, Cura, Meshmixer

This 3D print is a mesh of manmade objects placed in exaggerated positions with exaggerated sizes. The way 3D printing softwares Cura and Meshmixer generated supports for 3D printing mimicked the way in which nature grows. Both strive for developing strong and efficient “branches” that are ideal for growth. In this project I wanted to take manufactured objects that are built to specific requirements and see how the softwares would build up supports for the objects. I wanted to explore how far I can go in manipulating the generated supports based on the orientation of the objects as well as mixing the supports for two different softwares.
